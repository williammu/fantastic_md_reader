## 第二节：问题描述

这是第二个章节的内容。之前目录链接点击后没有反应，经过多轮调试发现以下问题：

### 2.1 问题一：空格处理

原始目录链接格式：
```
[第一节：测试概述](#第一节测试概述)
```

部分文档中可能包含空格：
```
[第一节：测试概述](#第一节测试概述 )
```

### 2.2 问题二：大小写不匹配

标题 ID 会被转换为小写：
- 标题：`## 第一节：测试概述`
- 生成 ID：`#第一节测试概述`（小写）
- 目录链接：`#第一节测试概述`（原始大小写）

## 第三节：解决方案

这是第三个章节的内容。针对上述问题，采用了以下解决方案：

### 3.1 修复空格处理

修改正则表达式，支持 # 号前后有空格的情况：
```javascript
/\[([^\]]+)\]\(\s*(#\S+)\s*\)/g
```

### 3.2 修复大小写匹配

在 JavaScript 中将锚点 ID 转换为小写：
```javascript
var anchorId = href.substring(hashIndex + 1).toLowerCase();
```
