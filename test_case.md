# MD 阅读器 - 测试用例文档

## 测试说明

本文档用于测试 MD 阅读器对所有 Markdown 元素的支持情况。每个测试用例包含：
- **输入**: Markdown 源码
- **预期结果**: 渲染效果描述
- **测试重点**: 需要验证的功能点

---

## 一、基础文本元素

### TC-001 普通段落
**输入:**
```markdown
这是一个普通段落。这是同一行内的文本。

这是另一个段落，与前一个段落有空行分隔。
```

**预期结果:**
- 两行文本正确渲染为两个独立段落
- 段落间距正常
- 自动换行功能正常

**测试重点:** 
- 段落间距、字体大小应用（默认54sp）、换行处理

---

### TC-002 粗体文字
**输入:**
```markdown
这是**粗体文字**测试。
这也是__粗体文字__测试。
```

**预期结果:**
- "粗体文字"以粗体显示
- 两种语法（** 和 __）效果一致

**测试重点:** 行内元素渲染、字体加粗

---

---
